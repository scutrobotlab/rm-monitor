# stt-normalize

`stt-normalize` scans existing `stt.jsonl` files and converts Traditional Chinese text to Simplified Chinese. It does not call whisper-server and does not touch audio or video files.

Examples:

```bash
stt-normalize -f /etc/rm-monitor/config.yml -dry-run -limit 100
stt-normalize -f /etc/rm-monitor/config.yml -match 31002
stt-normalize -f /etc/rm-monitor/config.yml -round 330 -no-report-reset
```

By default, changed matches have `matches.report` cleared so manifest/report and Lark cards can be regenerated by the existing pipeline.
